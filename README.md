# 电商图片尺寸

一键适配各电商平台图片尺寸的在线工具。

## 功能特性

- 🛒 **多平台支持**：淘宝/天猫、京东、拼多多、小红书、抖音、微信公众号
- 📐 **预设尺寸**：主图、详情页、轮播图、封面图等常用尺寸
- ✂️ **三种裁剪模式**：覆盖(Crop)、适应(Fit)、拉伸(Stretch)
- 📦 **批量处理**：一次选择多张图片，全部调整后批量下载
- ⚙️ **自定义尺寸**：支持输入任意宽高
- 🔒 **隐私安全**：纯前端处理，图片不会上传至服务器

## 技术栈

- 纯静态 HTML
- Tailwind CSS (CDN)
- Canvas API（图片裁剪/缩放）

## 部署

本项目部署在 [Cloudflare Pages](https://img-size.pages.dev/)。

### 本地开发

```bash
# 克隆仓库
git clone https://github.com/hpennn/img-size.git
cd img-size

# 直接用浏览器打开 index.html 即可
```

### Cloudflare Pages 部署

1. Fork 本仓库
2. 登录 [Cloudflare Dashboard](https://dash.cloudflare.com/)
3. 进入 Workers & Pages → Create → Pages → Connect to Git
4. 选择仓库，构建配置留空（纯静态站点）
5. 部署完成
